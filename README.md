# Polymorphic Portfolio

> *"It changes as you observe it."*

This is a personal web space for **Chris Melson**, powered by **Next.js 14**, **Tailwind CSS**, and **Framer Motion**. It features a "Trimodal" user interface that completely transforms the visual and semantic context of the site based on the selected persona.

![Portfolio Preview](./public/aboutme_portfolio.png)

## The Three Modes

1.  **The Executive (Default):**
    *   **Theme:** Global Operations, Annual Reports, High Contrast White/Navy.
    *   **Context:** Professional resume, career highlights, and leadership KPIs.
    
2.  **The Strategist:**
    *   **Theme:** Tabletop Gaming, MTG Artifacts, Warm Parchment/Ink.
    *   **Context:** Hobbies presented as "deck building," projects as "artifacts" or "cards."

3.  **The Engineer:**
    *   **Theme:** Terminal, Hacker Green/Black, Monospace.
    *   **Context:** Home Lab configs, raw JSON data, and "beating the system" aesthetic.

## Tech Stack

*   **Framework:** Next.js 14 (App Router)
*   **Styling:** Tailwind CSS + CSS Variables (Hot-swappable themes)
*   **Animation:** Framer Motion (Layout transitions & Intro sequence)
*   **Icons:** Lucide React
