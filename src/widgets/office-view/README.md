# Office View Widget

## Description
The **Office View** allows users to explore Christopher Melson's professional history through the lens of an "Operational Architect." It supports two primary modes:
1.  **Executive (Boardroom):** High-level strategic overview.
2.  **Strategist (Architect):** Mid-level operational details.
